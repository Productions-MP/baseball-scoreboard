Place licensed Futura font files here before running the Raspberry Pi installer.

Recommended filenames for the web app:

- `Futura-Book.woff2`
- `Futura-Book.woff`
- `Futura-Book.ttf`
- `Futura-Book.otf`
- `Futura-Bold.woff2`
- `Futura-Bold.woff`
- `Futura-Bold.ttf`
- `Futura-Bold.otf`

The installer copies any `.ttf` and `.otf` files from this folder into the Pi user's local fonts directory and refreshes the font cache.
