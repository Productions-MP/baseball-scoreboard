Place licensed Futura font files here before running the Raspberry Pi installer.

Recommended filenames for the web app:

- `Futura-Std-Medium.woff2`
- `Futura-Std-Medium.woff`
- `Futura-Std-Medium.ttf`
- `Futura-Std-Medium.otf`
- `Futura-Std-Bold.woff2`
- `Futura-Std-Bold.woff`
- `Futura-Std-Bold.ttf`
- `Futura-Std-Bold.otf`

Optional fallback filenames if you are using Light instead of Medium:

- `Futura-Std-Light.woff2`
- `Futura-Std-Light.woff`
- `Futura-Std-Light.ttf`
- `Futura-Std-Light.otf`

The installer copies any `.ttf` and `.otf` files from this folder into the Pi user's local fonts directory and refreshes the font cache.
